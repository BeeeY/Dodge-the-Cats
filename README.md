# Dodge the Cats

A game where you are a mouse trying not to get eaten. Cats appear on screen and the objective is to not touch them, which kills you, use wasd or arrow keys to stay alive for as long as possible. 

Demo : https://meek-choux-82cd55.netlify.app/

![alt text](https://github.com/BeeeY/Dodge-the-Cats/blob/main/Screenshot%202026-07-17%20115516.png)
## Built With
Game-Godot and gdscript

Art-Piskel

## Installing
1.Download Godot if not installed

2.Download the new-game-project folder

3.Open Godot

4.Press Import in the projects tab

5.select the new-game-project

6.Press F5 or the run game button to run the game.
